Безопасные типы данных Ещё один подход к синхронизации данных в распределённых системах — ис- пользование безопасных типов данных (Conﬂict-free Replicated Data Types, CRDT). Безопасными называются структуры данных, для которых определены правила разрешения конфликтов, не зависящие от логики приложения.
