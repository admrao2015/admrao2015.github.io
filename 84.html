3.2. Изменяемые структуры PCTFREE страница будет вновь считаться свободной только тогда, когда объ- ём свободного пространства превысит PCTUSED. Очевидно, что если данные в таблицу загружаются один раз, то имеет смысл задать параметр PCTFREE 0, но если приложение предполагает изменение данных, то лучше оставить в каждой странице запас свободного пространства. Сжатие данных У пользователей базы данных возникает естественное желание уменьшить объём занимаемого пространства, особенно если узким местом в работе явля- ются операции ввода-вывода. Существует два подхода к сжатию данных: • сжатие блоков стандартными алгоритмами (например, из библиотеки zlib); • словарное сжатие данных внутри блока с использованием алгоритмов се- мейства LZ. Разные реализации первого подхода используются в MySQL и Postgres Pro Enterprise — версии PostgreSQL, поставляемой компанией Postgres Professional. Postgres Pro 1 записывает сжатые страницы вплотную друг к другу, выравнивая каждую страницу по границе блока СХД.
